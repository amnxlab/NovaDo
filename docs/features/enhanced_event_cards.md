# Event Cards & Theming

NovaDo uses a modern, theme-aware design system for event cards to ensure readability and aesthetic appeal across all views.

## 🎨 Theme System

The application supports three distinct themes. Event cards adapt their colors, shadows, and gradients automatically.

### Light Theme
*Clean & Professional*
-   **Primary**: Soft white-to-gray gradients.
-   **Accents**: Subtle pastel colors for priorities.
-   **Shadows**: Gentle drops for depth.

### Dark Theme
*Sleek & Modern*
-   **Primary**: Deep slate gradients.
-   **Accents**: Rich, saturated colors.
-   **Contrast**: Enhanced white text for readability.

### Hacker Theme
*Cyberpunk / Matrix*
-   **Primary**: Black with green glow.
-   **Font**: Monospace integration.
-   **Effects**: Neon borders and glows.

## 🃏 Card Types

### Task Status
-   **Completed**: Green/Success tones with reduced opacity.
-   **In Progress**: Amber/Warning tones with a subtle pulse.
-   **Skipped**: Red/Error tones.
-   **Scheduled**: Default theme colors.

### Google Calendar Events
External events from Google Calendar retain their original color coding but are enhanced with the active theme's texture (glassmorphism/gradients) to blend seamlessly with the app.

## 👁️ Visual Design

-   **Glass Morphism**: Background blur effects on supported browsers.
-   **Typography**: Inter (sans-serif) for UI, Monospace for time displays.
-   **Responsive**:
    -   *Month View*: Compact bars.
    -   *Week/Day View*: Full detailed blocks.
    -   *Agenda View*: Large, list-style cards.
